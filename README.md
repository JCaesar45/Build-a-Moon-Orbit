# 🌕 Moon Orbit Animation

This project simulates a simple Moon orbiting the Earth using **HTML** and **CSS animations**. It's built from scratch and styled with a clean personal touch to demonstrate core web animation concepts using **Flexbox**, **positioning**, and **@keyframes**.

---

## 🚀 Live Demo

View the animation by opening `index.html` in any modern browser.

---

## 📁 Project Structure

``

moon-orbit-animation/
├── index.html        # Main HTML structure
├── styles.css        # Styling and animation
└── README.md         # Project documentation

``

---

## ✅ User Stories & Features

- [x] Page content is centered with Flexbox
- [x] Earth is centered in a 200x200 `space` container
- [x] Moon orbits Earth in a circular path
- [x] Uses `@keyframes` for smooth 360° rotation
- [x] Fully responsive to viewport
- [x] Clean and customized style

---

## 💻 How to Run

1. **Clone or download** this repository.
2. Open the `index.html` file in a web browser.

---

## 🛠 Technologies Used

- HTML5
- CSS3 (Flexbox, Positioning, Transforms, Keyframes)

---

## 🎯 Key Concepts

- Centering with Flexbox and `translate()`
- Absolute vs. relative positioning
- Circular motion with `transform: rotate()`
- Infinite animation with `@keyframes`
